## Goal

Keep the HTTP server bind and every client caller on 127.0.0.1:1234 as a single hardcoded value so shell hooks, Claude Code hooks, the MCP sidecar, and the Rust backend all agree without a discovery step.

## Rule

Any code that binds or talks to the Ani-Mime local HTTP server MUST use the literal `127.0.0.1:1234` (or `http://127.0.0.1:1234` for URLs). No environment variables, no CLI flags, no config files.

## Golden Example

```rust
// src-tauri/src/server.rs
let server = Server::http("127.0.0.1:1234").expect("bind 127.0.0.1:1234");
```
```zsh
# src-tauri/script/terminal-mirror.zsh
curl -s --max-time 1 "http://127.0.0.1:1234/status?pid=$$&state=busy&cmd=$cmd" >/dev/null 2>&1
```
```js
// src-tauri/mcp-server/server.mjs
const res = await fetch("http://127.0.0.1:1234/mcp/say", { method: "POST", body });
```
## Not This

| Anti-Pattern | Correct | Why Wrong Here |
| --- | --- | --- |
| Server::http("0.0.0.0:1234") | Server::http("127.0.0.1:1234") | Binds the world — Ani-Mime is strictly loopback-only |
| curl http://localhost:1234/... | curl http://127.0.0.1:1234/... | localhost may resolve to IPv6 ::1 and miss the IPv4 bind |
| let port = env::var("ANI_MIME_PORT")... | hardcoded 1234 | Shell scripts and MCP server have no access to env, so any override breaks the contract |
## Scope

Applies to src-tauri/src/server.rs, every shell hook under src-tauri/script/, src-tauri/mcp-server/server.mjs, and any Claude Code hook entry in ~/.claude/settings.json generated by setup/claude.rs. Does not apply to unrelated network code (none exists today).
