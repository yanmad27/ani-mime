## Goal

Reserve pid=0 as the single virtual Claude Code session so Claude's tool-use hooks can report busy/idle transitions without ever being evicted by the watchdog or mistaken for a real shell session.

## Rule

Any code that handles `/status` requests, iterates `AppState.sessions`, or prunes stale sessions MUST treat pid=0 as a reserved virtual session — never evicted by the watchdog, never subject to the 40-second stale-session timeout, and never confused with a real OS process. Only Claude Code hooks (PreToolUse, UserPromptSubmit, Stop, SessionStart, SessionEnd) emit pid=0.

## Rule

pid=0 is the reserved Claude Code virtual session. The watchdog never evicts it, status resolution treats it like any other session for priority, and shell hooks never emit it.

## Golden Example

```rust
// src-tauri/src/watchdog.rs
fn prune_stale(state: &mut AppState, now: u64) {
    state.sessions.retain(|pid, sess| {
        if *pid == 0 {
            return true; // Claude Code virtual session — never evicted
        }
        now - sess.last_seen < STALE_SESSION_SECS
    });
}
```
```zsh
# src-tauri/script/terminal-mirror.zsh — real shell sessions only
curl -s --max-time 1 "http://127.0.0.1:1234/status?pid=$$&state=busy" >/dev/null 2>&1
```
```bash
# ~/.claude/settings.json hook command — Claude Code virtual session only
curl -s --max-time 1 "http://127.0.0.1:1234/status?pid=0&state=busy" >/dev/null 2>&1
```
## Not This

| Anti-Pattern | Correct | Why Wrong Here |
| --- | --- | --- |
| `state.sessions.retain( | _, s | now - s.last_seen < 40)` |
| Shell hook emitting pid=0 | Use $$ (the real shell PID) | Collides with Claude Code and breaks the single-virtual-session contract |
| Looking up the Claude session by parent PID | Look up sessions.get(&0) | Claude has no stable PID from the backend's perspective — only the reserved 0 |
## Scope

Applies to src-tauri/src/state.rs, src-tauri/src/watchdog.rs, src-tauri/src/server.rs, every shell hook in src-tauri/script/, every Claude Code hook command generated by src-tauri/src/setup/claude.rs, and the MCP server at src-tauri/mcp-server/server.mjs.
